nastest

纯 Shell 实现的 NAS 多客户端并发压力测试工具
专为 MetroCluster 双活存储性能验证设计

一键安装 (需要 root 权限)
curl -fsSL https://raw.githubusercontent.com/NetApptool/nastest/main/get-nastest.sh | sudo bash
自动完成: 下载 → 解压 → 安装 → 启动交互式配置向导
GitHub 仓库 ⬇ 离线包下载 (8.4 MB)
🧙

交互式配置向导

问答式引导完成所有配置, 不需要手动编辑文件. 每填一台 Worker 立即验证 SSH 连通性.

⚡

多客户端并发

同时在多台 Worker VM 上执行 fio 压测, 实时进度条显示测试进度和剩余时间.

⚖️

双站点对比

Site A / Site B 性能均衡性分析, 专为 MetroCluster 双活架构设计, 自动计算偏差百分比.

📊

HTML 可视化报告

自包含单文件, 浏览器直接打开. 深蓝色银行风配色, 支持深色/浅色切换.

💬

面向非技术人员

自动生成通俗易懂的测试结论, 用银行柜台、高速公路等比喻解释 IOPS、延迟等术语.

📦

零依赖部署

内置 fio 3.37 + iperf3 3.20, 无需目标机器安装额外软件. 离线环境开箱即用.

🐧

麒麟 Linux 兼容

专为银行客户离线麒麟 Linux 环境设计, 纯 Shell 实现, 不依赖 Python 或任何运行时.

🔧

灵活测试模式

随机读写(数据库)、顺序读写(备份)、混合读写(文件服务) 三种模式, 可自定义时长.

./nas_bench.sh setup

# 使用已有配置开始测试
./nas_bench.sh start
./nas_bench.sh start --mode seq --duration 600

# 检测连通性 / 清理数据
./nas_bench.sh check
./nas_bench.sh clean
